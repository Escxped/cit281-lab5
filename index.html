🧪 Lab 5 – Express Server Routes

This lab expanded on Express fundamentals, with routes for GET and POST requests. It included building a server that could return all students, a single student, and handle unmatched routes. We also added POST functionality to add a new student via JSON.

🔍 Topics Covered

  • Route handling with Express
  • Route parameters and dynamic IDs
  • Parsing JSON with express.json()
  • Handling unmatched routes
  • POSTing JSON data to the server
